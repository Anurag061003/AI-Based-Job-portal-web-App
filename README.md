# AI-Based Job Portal Web App

## 📌 Overview
This project is a full-stack **Job Portal Application** built using the MERN stack (MongoDB, Express.js, React, Node.js).  
It provides job seekers with AI-powered resume scoring and personalized job recommendations, while employers can post and manage job listings.

---

## 🛠️ Tech Stack
- **Frontend:** React, HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT, bcrypt
- **AI Integration:** OpenAI API for resume scoring
---
## Setup Backend 
cd backend
npm install
npm run dev

## Setup frontend 
cd frontend
npm install
npm start
## 🔑 Environment Variables

### Backend (`/backend/.env`)
Required variables:
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
OPENAI_API_KEY=your_openai_api_key
